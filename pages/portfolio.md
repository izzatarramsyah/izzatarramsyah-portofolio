# Portofolio

## Angular + Spring Boot Project (Practice Project)

**Description** : Web Application for Growth Monitoring According Z Score WHO Study Case

**Stack** : Java Spring Boot, Angular, PostgreSQL

**Repo** : 
- Frontend : https://github.com/izzatarramsyah/biyubi-clinic-frontend-admin
- Backend : https://github.com/izzatarramsyah/biyubi-clinic-backend
- Cron Job/Scheduler : https://github.com/izzatarramsyah/biyubi-clinic-backend-scheduler

**Feature** : 

- Input or Upload (Excel) Data Master Vaccine and Medical 
- Input Data Vaccine and Medical Record
- Generate Vaccine and Medical Schedule and Report
- Show Grapic of Growth based on Weight, Height and Head Diameter
- Send SMS/Whatsapp Vaccine and Medical Checking Reminder by Scheduler

- Queue Information
